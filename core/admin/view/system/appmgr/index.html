什么是应用？

应用是指基于ShopEx商店系统所开发的第三方程序功能，譬如整合淘宝的一系列功能，您可以点击下面按钮去挑选所需要的应用，然后安装即可开始使用。

安装应用的好处？

安装应用后您可以更方便、更快捷地管理您的网店，提升工作效率，例如同步管理淘宝商品、处理订单、下载记录等让您在ShopEx上跨平台处理业务，方便省事，高效便捷！
